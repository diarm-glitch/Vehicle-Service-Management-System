# Vehicle Service Management System - Frontend

## Technologies Used

- React
- Vite
- Axios
- Bootstrap

## Installation

1. Open terminal in Frontend folder

npm install

2. Start development server

npm run dev

## Features

- User Authentication
- Role-Based Access Control
- Dashboard
- Vehicle Management
- Customer Management
- Appointments Management
- Parts Management
- Invoices Management
- Car Reviews
- Buy & Sell Cars
- Emergency Pickup Requests
- Mechanic Applications
- Roadside Assistance Subscriptions

## User Roles

### Admin
- Full access

### Staff
- Service management
- Customer management

### User
- Profile management
- Vehicle tracking
- Reviews
- Sell vehicles

### Premium User
- All User features
- Premium roadside assistance
